Nome do elemento: Viagem
Descrição: É um sistema para facilitar o gerenciamento de pacotes de viagem, permitindo as agências registrarem destinos, preço, e transporte, além disso, os usuários podem planejar férias incríveis e comparar os pacotes para encontrar a melhor oferta possível.
Dados:

id: inteiro
destino: texto
precoPacote: real
duracaoDias: inteiro
tipoTransporte: char ('A' para avião, 'B' para ônibus)

Instruções
Nessa atividade será entregue a versão de Pilha com apontadores.

As funções dessa atividade:

criarPilha
push (insere no topo da pilha) 
pop (remove do topo da pilha) 
ehVazia (informa se a pilha está vazia) 
verTopo (informa o conteúdo do elemento do topo da pilha)
excluirPilha (apaga todos os elementos e a pilha)
carregarDados (realiza a leitura dos dados presentes no arquivo)
salvarDados (armazena os dados presentes na lista no arquivo)
main (usando como uma aplicação e chamando todas as funções anteriores)
