Nome do elemento: Viagem
Descrição: É um sistema para facilitar o gerenciamento de pacotes de viagem, permitindo as agências registrarem destinos, preço, e transporte, além disso, os usuários podem planejar férias incríveis e comparar os pacotes para encontrar a melhor oferta possível.
Dados:

id: inteiro
destino: texto
precoPacote: real
duracaoDias: inteiro
tipoTransporte: char ('A' para avião, 'B' para ônibus)
